# Roadmap MVP · 12 meses

## Fase 0 (Semanas 1-2)
- Discovery legal/fiscal/operativo
- Definición de KPIs y North Star
- Diseño UX base y lenguaje simple

## Fase 1 (Meses 1-3)
- Landing institucional + registro
- Auth JWT + Roles
- Perfiles 360 básicos
- Membresías Basic/Pro/Elite
- Comunidad inicial (foros + chat)

## Fase 2 (Meses 4-6)
- Módulo legal y administrativo guiado
- Facturación y contratos base
- Motor gamificación v1 (misiones, puntos, logros)
- Marketing interno v1 (promos y afiliados)

## Fase 3 (Meses 7-9)
- Marketplace completo (70/30)
- Ordenes, pagos, comisiones
- Inventario y logística + dropshipping
- Dashboard administrativo y auditoría

## Fase 4 (Meses 10-12)
- Espacios 3D iniciales (showrooms/eventos)
- Conciertos virtuales y avatar access
- Escalamiento cloud y observabilidad completa
- Programa enterprise y alianzas estratégicas
