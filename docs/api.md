# API (resumen operativo)

Este documento complementa `api-spec.md` con agrupación por carpetas de rutas propuesta:

- `routes/auth.js`: registro, login, refresh JWT.
- `routes/users.js`: perfil del usuario, roles y settings.
- `routes/memberships.js`: planes, upgrades, renovaciones.
- `routes/projects.js`: marca/proyecto/empresa/perfil artístico.
- `routes/products.js`: catálogo marketplace y órdenes.
- `routes/legal.js`: checklist fiscal, contratos, facturación.
- `routes/gamification.js`: misiones, logros, ranking.
- `routes/community.js`: foros, chats y mentorías.
