# AI Apps Open Source para correr local (Hugging Face)

## Objetivo
Agregar capacidades IA locales para marketing, comunidad, audio y assets visuales sin depender 100% de servicios externos.

## Apps sugeridas (MVP)
1. **Text Generation Inference (TGI)**
   - Uso: asistente interno, generación de copies, FAQs.
2. **Transformers + Gradio App**
   - Uso: clasificación de contenido, resumen de foros/chats.
3. **Whisper ASR (open source)**
   - Uso: transcripción de audio/video de mentorías y conciertos.
4. **Diffusers (Stable Diffusion)**
   - Uso: creativos para campañas, portadas y merch.

## Exposición interna
- API proxy: `/api/v1/ai-apps/*`
- Acceso por rol/membresía (Pro y Elite)
- Logging y cuotas por usuario
